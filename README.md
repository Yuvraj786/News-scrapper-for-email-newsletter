# News-scrapper-for-email-newsletter
The News-scrapper is a python program to send Email of latest news & top-stories scrapped from various sites, It uses geckodriver, BeautifulSoup to access the websites.

Build Status Known Vulnerabilities codecov.io Code Coverage

Outlook mail provision added!

Libraries Used
Selenium
smtplib
BeautifulSoup
gtts
Driver & Browser
Geckodriver (driver)
Mozilla Firefox (browser--latest version)
Installation
pip install selenium
pip install secure-smtplib
pip install gTTS
pip install beautifulsoup4
